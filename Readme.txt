2人での対戦型のゲームに変更しました。
画面は合計3つあります。
①main
②wait
③socket

mainではroom(int)を指定しroomを作成しwaitに移行します。
waitでは参加者が1人では待機画面になり、参加者が2人になるとゲーム画面(socket)に移行します。
socketでは数当てゲームができます。ゲームの内容としては同じです。

使用技術
・Flask, MySQL, SocketIO, HTML, CSS, JavaScript, Docker

改善するべきバグ
・waitで人数が2人になった時に1人目の画面が自動で切り替わらない
・交互に入力させる動作はsocket通信を用いており最初のページロード時ではそれが、
　動作しないので場合によっては、ユーザーの入力できる数が均等にならない。

改善すべき箇所
・socket画面での入力を半角の数に限定していない
